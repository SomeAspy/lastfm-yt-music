## Requirements
- A jailbroken device OR the YouTube Music IPA sideloaded with this tweak.

## Setup
- Fill out the `LASTFM_API_KEY` and `LASTFM_API_SECRET` variables in `Makefile`
- Build with `[make|gmake] clean package FINALPACKAGE=1`
